# SILAKAP Web

Frontend SILAKAP berbasis React, Vite, Tailwind, dan komponen UI yang sudah diserap ke dalam aplikasi.

## Development

```bash
npm run dev:web
```

Default lokal:

```text
http://localhost:5173
```

API lokal:

```text
http://localhost:3101/api/v1
```

Konfigurasi ada di:

```text
apps/web/.env
apps/web/.env.production
```

## Build

```bash
npm run build:web
```

Output build:

```text
apps/web/dist
```
