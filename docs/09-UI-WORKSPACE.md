# UI Workspace

## 1. Principle

UI berbasis pekerjaan harian.

Bukan menu-oriented, tetapi work-oriented.

---

## 2. Workspace ASN

- submit layanan
- upload dokumen
- tracking status
- melihat catatan revisi

---

## 3. Workspace OPD

- verifikasi ASN
- monitoring unit
- revisi layanan
- submit ke BKPSDM

---

## 4. Workspace PPPK

- checklist dokumen
- arsip pending
- validasi file
- monitoring dokumen invalid

---

## 5. Workspace Pertama

- validasi awal
- pending verification
- return administrasi
- checklist layanan

---

## 6. Workspace Muda

- telaah substansi
- validasi layanan
- assign task substansi
- catatan rekomendasi

---

## 7. Workspace Madya

- QC
- monitoring SLA
- audit kualitas
- review kasus berisiko

---

## 8. Workspace Kabid

- approval pending
- bottleneck
- workload staf
- SLA merah
- dashboard bidang

---

## 9. Standard Layout

```text
Header
↓
Case Summary
↓
Task Workspace
↓
Document Workspace
↓
Timeline
↓
Action Panel
```

## 10. UI Component Standard

- status badge

- workflow timeline
- task card
- document preview
- action panel
- empty state
- loading state
- audit timeline
