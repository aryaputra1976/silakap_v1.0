# SILAKAP Hostinger API

Folder ini dibuat agar cocok dengan pengaturan Hostinger pada dashboard:

```text
Preset framework : NestJS
Branch           : main
Versi node       : 18.x
Root directory   : api
Build command    : npm run build
Direktori output : dist
Entry file       : main.js
```

Endpoint awal:

- `GET /health`
- `GET /api/v1/sidata/asn`
- `GET /api/v1/siap/cases`
- `GET /api/v1/siap/tasks`
- `GET /api/v1/sipensiun/cases`
- `GET /api/v1/siarsip/documents`
- `GET /api/v1/analytics/dashboard`
